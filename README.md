# Kotlin Absensi (Maxmar Attendance)

Native Android attendance app with Jetpack Compose, connecting to Laravel backend.

## Tech Stack
- Jetpack Compose (UI)
- Hilt (DI)
- Retrofit (API)
- CameraX + ML Kit (Face Detection)
- Google Maps (Location)
- Firebase FCM (Push)

## Backend
Laravel API: `filament_absensi`
